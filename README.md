# 坦克大战

经典 HTML5 Canvas 坦克大战游戏，支持浏览器直接运行。

## 在线游玩

[开始游戏](https://xinmiaoyin.github.io/tank-battle/)

## 操作说明

| 按键 | 功能 |
|------|------|
| W / A / S / D | 上 / 左 / 下 / 右移动 |
| 方向键 | 上 / 左 / 下 / 右移动 |
| 空格键 | 射击 |

## 游戏特性

- **玩家坦克** — 绿色，可收集强化（提升子弹穿透力）
- **三种敌人**：
  - 红色 — 普通敌人
  - 橙色 — 快速敌人
  - 紫色 — 重装敌人（需要 2 发子弹才能消灭）
- **四种地形**：
  - 砖墙 — 可被子弹摧毁
  - 钢墙 — 普通子弹无法摧毁，强化子弹可击穿
  - 水域 — 无法通过，有动态波纹效果
  - 树林 — 可穿过，提供视觉遮挡
- **基地保护** — 底部带 ★ 的方块为基地，被摧毁则游戏结束
- **多关卡递进** — 每关敌人数量递增，墙体密度增大

## 本地运行

下载 `index.html` 文件后直接用浏览器打开即可。

## 技术栈

- 原生 HTML5 Canvas + JavaScript，无任何外部依赖
